# Specification

## Summary
**Goal:** Create a professional landing website for GigMarket, a platform connecting local workers with employers, featuring 5 separate dashboard pages accessible via navigation cards.

**Planned changes:**
- Build landing page with hero section, "How It Works" steps, features section, worker categories grid, stats, trust badges, testimonials, and CTA footer
- Create 5 navigation cards linking to separate dashboard pages: About Application, User Manual, Review Centre, Complaint Box, and Download APK
- Implement About Application dashboard with mission, problem statement, target users, benefits, screenshots, and founder info
- Implement User Manual dashboard with account creation, job posting, worker application, hiring process, safety tips, and FAQ accordion
- Implement Review Centre dashboard with ratings summary, review cards, worker/employer filter, write review form, and analytics chart
- Implement Complaint Box dashboard with complaint form, category dropdown, file upload, ticket status UI, and previous complaints list
- Implement Download APK dashboard with download button, version info, screenshots carousel, installation steps, and Play Store coming soon section
- Set up client-side routing for navigation between landing page and 5 dashboards
- Apply modern startup UI design with soft shadows, rounded cards, blue/green trust colors, accessible fonts, and mobile-first responsive layout
- Populate all sections with realistic dummy data for demo purposes

**User-visible outcome:** A fully navigable, demo-ready website showcasing GigMarket's value proposition with landing page and 5 functional dashboard pages, styled professionally for low-tech users and optimized for mobile viewing.
